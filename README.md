# Sistem Inventori Sparepart Bengkel
nama: valysa azzahra 
NIM: 1323071

Aplikasi ini dibuat sebagai **Tugas Praktikum Pemrograman Web**.  
Sistem digunakan untuk membantu pengelolaan stok sparepart bengkel, mulai dari pencatatan data sparepart, transaksi stok masuk dan keluar, hingga laporan stok.

## Fungsi Sistem
- Mengelola data sparepart, kategori, dan supplier
- Mencatat stok masuk dan stok keluar
- Menyajikan laporan stok dan transaksi

## Role Pengguna
- **Admin**: kelola data, transaksi, dan user
- **Staff**: kelola data dan transaksi
- **Owner**: melihat laporan dan dashboard

Sistem menerapkan **Role-Based Access Control (RBAC)**.
